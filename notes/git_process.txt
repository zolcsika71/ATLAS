Clone Your Forked Repository:

Open PyCharm and select Get from VCS on the welcome screen, or if you have a project open, go to File -> New -> Project from Version Control.
In the URL field, enter the URL of your forked repository (you can get this URL from your repository page on GitHub).
Specify the local directory where you want to clone the repository and click Clone.

Open the Cloned Repository as a Project:

After cloning, PyCharm will ask if you want to open the directory as a project. Confirm this to proceed.
